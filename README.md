# Backend Assignment – Events API

This repository contains the backend implementation for Event-related APIs (Task 1).

## Tech Stack
- Node.js
- Express
- MongoDB (Native Driver)

## Running the Project
1. Install dependencies: `npm install`
2. Create a `.env` file with MongoDB credentials
3. Start the server: `npm start`
